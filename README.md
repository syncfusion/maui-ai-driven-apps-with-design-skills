# AI‑Driven UI Building for .NET MAUI
## Design Systems & UI Building Skills

This repository contains **reference implementations, patterns, and sample resources** for building **consistent, scalable UI** in **.NET MAUI** using **design systems** and **UI building skills**, with a focus on **AI‑assisted UI composition**.

The goal is to demonstrate how **structured design systems + skills** provide the **missing context** that AI needs to build reliable, production‑ready user interfaces.

---

## 🎨 Design System (MAUI‑Friendly)

The design system defines **UI standards and constraints** used across the application.

### Includes
- Design tokens (colors, spacing, typography)
- Reusable components
- Layout patterns and guidelines

Example:

```xml
<Color x:Key="ColorPrimary">#2563EB</Color>
<Thickness x:Key="SpacingMD">16</Thickness>
```

---

## 🧠 Skills (UI Building Intelligence)

Skills are **programmable abstractions** that teach AI **how to apply the design system correctly**.

> **Skills do not render UI — they constrain UI decisions.**

---

## 🤖 AI‑Driven UI Workflow

1. Developer provides intent
2. AI engine interprets intent
3. Skills provide context and constraints
4. UI composition is assembled
5. MAUI renders the final UI
